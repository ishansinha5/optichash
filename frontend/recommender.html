LIVE MATCH ENGINE

I wanted to see if I could build a computer vision pipeline that didn't immediately melt my laptop's CPU. The goal here is efficient routing. When you upload a messy photo of a comic book below, the system doesn't instantly fire up a heavy neural network.

Instead, it routes the image to a native C++ microservice that acts as a bouncer, calculating a byte-level Perceptual Hash (pHash). If it finds an exact spatial match in the database, it returns the metadata instantly—saving massive amounts of compute. Only if it's a completely new, unrecognized image does the system pass the request to the PyTorch machine learning layer to make an active prediction.
