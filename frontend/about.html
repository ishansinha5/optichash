SYSTEM ARCHITECTURE

The "Green AI" Constraint:
Standard image recognition burns massive amounts of cloud compute. This engine was built as an exercise in hardware-aware constraints.

1. Spatial Caching (C++):
Images are evaluated instantly by a native C++ Perceptual Hash bouncer. Known covers skip machine learning entirely.

2. Quantized Edge Compute (PyTorch):
The fallback vision model uses a MobileNetV3 backbone heavily pruned and quantized down to 8-bit integers (INT8), running localized inference at lightning speeds without requiring a GPU.
