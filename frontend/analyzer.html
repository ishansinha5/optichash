This scanner represents the front-end of a hardware-aware computer vision pipeline optimized for edge deployments. The core focus is efficient system routing and sustainable execution.

When a photo is submitted, the payload is first routed to a native C++ microservice that calculates a geometric Perceptual Hash (pHash). If it finds an exact spatial match in the local cache, it returns the requested metadata instantly without invoking heavy compute.

If the cover is heavily obscured, distorted, or unrecognized by the cache, the system gracefully falls back to a highly quantized INT8 PyTorch engine. This neural network has been specifically trained on extreme edge cases—including harsh lighting glare, deep shadows, steep viewing angles, and real-world background messes—ensuring high-accuracy recognition in everyday environments.
